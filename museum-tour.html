Museum Tour

This 360-degree experience is a prototype developed as part of an academic assignment. The project is built based on an exhibition named "Breaking the Ice", created in partnership with Antarctic Heritage Trust. Breaking the Ice tells the story of Carsten Borchgrevink’s Southern Cross expedition through objects the explorers left behind in the two huts they built at Cape Adare. The artefacts include scientific equipment, clothing and sledging supplies.


The tour’s design has two modalities: “Explorer” and “Guided”. The explorer mode allows the users to navigate the exhibition without following a particular direction. This mode also allows the users to interact with 3D artefacts. On the other hand, the guided mode has a specific pathway and features a tour guide that provides more details about the artefacts’ history.

Working on this project provided me with new insights into how technology can be used to achieve stunning results, as many are the applications that 360-degree interactions offer. This prototype was developed for PC platforms using the Unity3D engine and the Insta360 camera.
