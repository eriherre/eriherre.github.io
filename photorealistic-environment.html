Photorealistic Environment

This project features a virtual reality environment where users can navigate through a photorealist scene while interacting with 3D objects. I developed this application as part of my learning journey. Maybe one of my first attempts into the world of game engines and immersive realities.


My essential tasks were designing the apartment and set up the ambient lighting. I was also responsible for configuring the navigation and interactions using a visual scripting tool (Blueprints). This project was developed using the Unreal Engine and the HTC VIVE Pro.
