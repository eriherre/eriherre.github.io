Delinker

This e-learning module was developed to help people learn how to operate machinery safely. The delinker is a sausage separating machine commonly used by many food manufacturers. It is not considered high-risk machinery, but still, as with many electrical devices with moving parts, only competent personnel is authorised to operate it.


The challenge here was to develop an interactive instruction covering fundamental safety aspects and functional stages of the machine without bombarding trainees with too much technical content. The main sections used to structure this module are:

  1. Set-Up, production is about to start, and trainees need to learn how to prepare their workstations and assemble all the parts.
  2. Operation, this section shows the basics of how to run the machine. Here we cover essential safety and quality aspects related to personnel and food, respectively.
  3. Disassemble, production is over, and trainees need to learn how to remove all the parts safely, so the machine gets ready for a good cleaning.
  4. Cleaning, this section represents the end of the workflow. Here trainees learn how to clean all the parts of the machine safely so it is ready to be used when production starts the next day.

Lastly, the course ends with a quiz to engage the audience and test knowledge retention. This e-learning module was developed using Storyline 360 and Adobe Premiere Pro for video editing.
