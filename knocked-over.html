Knocked Over

Knocked Over is a 3D bowling mini-game developed as an assignment for the section Counting Prototype, part of the Unity Junior Programmer Pathway. The requirements for this game to be successful were to design a simple application that counted something and a UI component that displayed that count.


The core mechanics are simple. Players can move and throw the bowling ball using the arrow and space keys. The UI component displays how many pins are knocked down. A “Strike” message and particle effects are shown at the centre of the screen if the player manages to knock down all the pins. Players can resume the game at any time by pressing a “restart” button. This mini-game was developed using the Unity3D engine, and all the assets were downloaded from the Unity Asset Store.
