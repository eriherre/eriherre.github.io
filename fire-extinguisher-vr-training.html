Fire Extinguisher VR Training

This project represents a virtual simulation where the main objective is to help users learn how to navigate through an immersive space and put out a fire using an extinguisher. The core sections are “Introduction”, “Navigation”, “Interactable Objects”, and “Practical Exercise.” That structure was created to help users build knowledge until they achieve the desired outcome.


A virtual human guides the instruction while the user explores the environment by teleporting and interacting with UI elements and 3D objects. Having a virtual instructor within the environment allows the experience to be more consistent and less disruptive. Moreover, the virtual controllers highlight their buttons to inform the user how to perform actions. Once the user has some opportunities to practise, the controllers are replaced by virtual hands to support the feeling of presence within the environment.

This VR simulation was developed using the Unity3D engine and the HTC VIVE Pro. The virtual human was designed using Adobe Fuse CC, and it was rigged and animated inside Unity with VIVE equipment. The agent’s voice was created using Microsoft Text-to-Speech. Lastly, the extinguisher was modelled using Autodesk Maya and Adobe Subtance 3D Painter for texturing.
